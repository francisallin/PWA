Instructions to use BVP calculator
1. Choose the power conversion mode
2. If Custom mode is chosen, input your original BVD and new BVD
3. Input the original Rx in diopter form (Eg: -3.75, +2 etc, not -375 or +200 ).
4. Click/tab on any white space of the form. The result will be displayed automatically
